# ESERCIZIO

Data la lista VIP per una serata, verifica se l'utente X è presente nella lista VIP o se è nella lista standard.

## RAGIONAMENTO

- Creo una lista con i nomi delle persone nella lista VIP
- creo una lista con i nomi delle perosne nella lista standard
- chiedo all'utente di inserire il suo nome
  - SE il nome dell'utente è nella lista VIP 
    - stampo "VIP"
  - ALTRIMENTI SE il nome dell'utente è presente nella lista standard 
    - stampo "Standard"
