# ESERCIZIO

Date le temperature settimanali, trova il valore massimo, il minimo e calcola la media aritmetica delle sette giornate.


## RAGIONAMENTO

- Creo un array con i giorni della settimana
- creo un array inserendo le rispettive temperature dei giorni
  - stampo la temperatura massima
  - stampo la temperatura minima
  - stampo la media aritmetica delle temperature della settimana
